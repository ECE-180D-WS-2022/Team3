# Team 3

# To Run

`python3 main.py`

# Requirements

run these commands to use the client

```sh
cd client
make run
```

to lint the python files

```sh
cd client
make lint
```

to update the dependencies

```sh
cd client
conda activate team3
conda install [something]
make update
```

to delete the team3 environment after you are done with this project

```sh
cd client
make remove-env
```
