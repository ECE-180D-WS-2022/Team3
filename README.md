# Team 3

# To Run

`python3 main.py`

# Requirements

run these commands to use the client
```sh
cd client
conda env create --file environment.yaml
conda activate team3
python3 main.py
```

to update the dependencies
```sh
cd client
conda intall [something]
conda env export > environment.yaml
```
